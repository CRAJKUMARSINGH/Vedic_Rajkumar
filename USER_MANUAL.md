## 🛠️ **Troubleshooting**

### **❓ Common Issues**

#### **Q: Chart calculations seem incorrect**
**A**: 
1. Verify birth data accuracy (date, time, location)
2. Check Ayanamsa setting (Lahiri Chitrapaksha recommended)
3. Ensure correct time zone
4. Try refreshing the page

#### **Q: Transits not updating for selected date**
**A**: 
1. Click the "Update" button after changing date
2. Clear browser cache
3. Check internet connection
4. Try a different browser

#### **Q: PDF export not working**
**A**: 
1. Disable popup blockers
2. Check browser permissions
3. Try Chrome or Firefox
4. Reduce chart complexity

#### **Q: App running slowly**
**A**: 
1. Clear browser cache and cookies
2. Close unnecessary tabs
3. Check internet speed
4. Use mobile app for better performance

### **🔧 Technical Solutions**

#### **Browser Compatibility**
- **Recommended**: Chrome 90+, Firefox 88+, Safari 14+
- **Mobile**: iOS 13+, Android 8+
- **Internet**: Stable connection required

#### **Performance Tips**
- Use latest browser version
- Enable hardware acceleration
- Close background applications
- Regular cache cleanup
